# Game Of Life

## Install

```shell
pipenv install "git+https://github.com/pygame/pygame.git#egg=pygame"
pipenv install numpy
```
or run `pip3 install -r requirements.txt`

## Execute

```shell
pypenv run python __main__.py
```
